Blockset について

Blockset は、航空宇宙、通信、無線周波数 (RF)、信号処理などのアプリケーションの特定の機能で Simulink を拡張します。

インストールの内容によっては、すべての Blockset を含んでいない可能性があります。 どの Blockset がインストールに含まれているかを確認するには、左ペインの [Blocksets] エントリを展開します。 そこから、デモを表示するために Blockset を選択し、次に表示して実行する個々のデモを選択してください。 [ヘルプ] 内の例も見ることができます。 [コンテンツ] ペイン内で、ドキュメンテーションからサンプル コードを表示、実行、またはコピーするには、製品を選択し、[例] を選んでください。

デモのある製品のリストを参照するには、www.mathworks.com/products/demos をご参照ください。
